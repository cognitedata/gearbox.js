# Hoverable Preview

`HoverablePreview` component is meant to be a generic display component appearing after hovering an question mark icon.
It has four static properties: Header, Title, CloseButton and Cell.
Cell accepts children - either strings, or React Components. 

## Examples

### Appearing on icon hover

<Preview>
  <Story name="With the hover functionality">
    <div style={{width: '500px', border: '1px solid #000', padding: '10px', display: 'flex', flexDirection: 'row'}}>
      Hover me => 
      <HoverablePreview displayOn='hover'>
        <HoverablePreview.Header>
          <HoverablePreview.Title>This component should appear on hover.</HoverablePreview.Title>
        </HoverablePreview.Header>
        <HoverablePreview.Cell title="A cell">
          Lorem ipsum dolor sit amet, consectetur adipiscing elit.
        </HoverablePreview.Cell>
      </HoverablePreview>
    </div>
  </Story>
</Preview>
