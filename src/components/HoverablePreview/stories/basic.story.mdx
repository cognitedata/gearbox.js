# Hoverable Preview

`HoverablePreview` component is meant to be a generic display component. It can be displayed either after hovering an question mark icon, after clicking the question mark icon, or be displayed by default.

It has four static properties: Header, Title, CloseButton and Cell.

Cell accepts children - either strings, or React Components. 

## Examples

### Always visible

<Preview>
  <Story name="Always visible">
    <HoverablePreview>
      <HoverablePreview.Header>
        <HoverablePreview.Title>This component should be always visible....</HoverablePreview.Title>
      </HoverablePreview.Header>
      <HoverablePreview.Cell title="...because..." align="right">
        ...it doesn't have the optional "displayOn" property.
      </HoverablePreview.Cell>
      <HoverablePreview.Cell title="displayOn='hover'">
        ...would cause this component to have an (?) icon attached, upon hovering which the component appears.
      </HoverablePreview.Cell>
      <HoverablePreview.Cell title="displayOn='click'" align="left">
        ...causes this component to appear after clicking on the attached (?) icon.
      </HoverablePreview.Cell>
    </HoverablePreview>
  </Story>
</Preview>
