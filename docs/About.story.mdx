Gearbox is a library of React.JS components for building web applications on top of the Cognite Data Fusion library.

</div>




| Component            | Role                                                         |
| -------------------- | ------------------------------------------------------------ |
| AssetBreadcrumb      | Breadcrub visualization of an asset's position in the hierarchy |
| AssetDetailsPanel    | Two column table visualization of an assets key details      |
| AssetDocumentsPanel  | A list of documents associated with an asset                 |
| AssetEventsPanel     | Four column paginated table visualization of events associated with an asset |
| AssetMeta            | All info on an asset                                         |
| AssetScanner         | Recognize assets from an image                               |
| AssetSearch          | Search asset by name                                         |
| AssetTimeseriesPanel | A list of timeseries associated with an asset                |
| AssetTree            | Multi-root tree visualization of assets                      |
| DescriptionList      | A generic table visualization for two column data            |
| EventPreview         | Display event details                                        |
| Model3DViewer        | View 3D models                                               |
| SensorOverlay        | An overlay with live labels for time series sensor data for use in infographics |
| SVGViewer            | View SVG asset diagrams with search/locate, zoom/pan         |
| TenantSelector       | A form for selecting a tenant. This can be used to implement a sign-in form |
| ThreeDNodeTree       | Tree visualization of 3D node hierarchy                      |
| TimeseriesChart      | Visualize time series                                        |
| TimeseriesChartMeta  | Enhances TimeseriesChart with period selection, current sensor value, and meta data |
| TimeseriesDataExport | Export TS data in CSV format                                 |
| TimeseriesPreview    | Display the current sensor value for a TS                    |
| TimeseriesSearch     | Search and select time series by name                        |
