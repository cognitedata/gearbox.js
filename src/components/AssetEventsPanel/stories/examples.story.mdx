## Handle callbacks

A callback function can be provided to fire when the asset is loaded:

```jsx
import { CogniteEvent } from '@cognite/sdk';

const handleAssetEventsLoaded = (events: CogniteEvent[]) => console.log(events);
```

<Preview>

      assetId={4650652196144007}
