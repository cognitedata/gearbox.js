# Timeseries

> WIP

`TimeseriesChart` component is used for visualizing one or time series.
It loads the datapoints given a timeseries id and renders a line chart 
of those points.

`TimeseriesChartMeta` - Enhances TimeseriesChart with period selection, current sensor value, and meta data

`TimeseriesDataExport` - Export TS data in CSV format

`TimeseriesPreview` - Display the current sensor value for a TS

`TimeseriesSearch` - Search and select time series by name
